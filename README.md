# AirPods Pro Listening Mode

Control your AirPods Pro's listening mode via Alfred Workflow.

Use `ap` + `1~3` to change your AirPods's listening mode:

- `1` for Transparency mode.
- `2` for Off mode.
- `3` for Noise Cancellation mode.

Or directly adjust the listening mode via individual keyword:

- `trp` for Transparency mode.
- `off` for Off mode.
- `anc` for Noise Cancellation mode.
